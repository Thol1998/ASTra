 18                  If ALL EDGES were requested, these messages are followed by an end message (response): COO END 
 19                  With A and B the edge extremities 
 20                   
 21                   
 22  EDGES LENGTH 
 23  (4) Get all edges length (request): LEN 
 24   
 25  (5) Get specified edges length (request): LEN edge1 edge2...edgeN 
 26   
 27  (6) Edges length (response): LEN edge1 length1...edgeN lengthN (N in [0-Y] if all edges were requested (See constants for Y value)) 
 28                  If ALL EDGES were requested, these messages are followed by an end message (response): LEN END 
 29                   
 30                   
 31  EDGES CONGESTION 
 32  (7) Get all edges congestion (request): CON 
 33   
 34  (8) Get specified edges congestion (request): CON edge1 edge2...edgeN 
 35   
 36  (9) Edges congestion (response): CON edge1 congestion1...edgeN congestionN (N in [0-Y] if all edges were requested (See constants for Y value)) 
 37                  If ALL EDGES were requested, these messages are followed by an end message (response): CON END 
 38                  With congestion = average speed on the edge during the last step / max speed on this step 
 39                   
 40   
 41  EDGES SUCCESSORS (GRAPH) 
 42  (10) Get the graph (each edge linked with each edge successor) (request): SUC 
 43   
 44  (11) Get some specified edges successors (request): SUC edge1 edge2...edgeN 
 45   
 46  (12) Edges successors (response): SUCC,edge1 succ1 succ11 succ12,edge2 succ21 succ22,...,edgeN succN1 succN2 (N in [0-Y] if all edges were requested (See constants for Y value))

 48                   
 49                   
 50  BLOCK/UNBLOCK EDGE
